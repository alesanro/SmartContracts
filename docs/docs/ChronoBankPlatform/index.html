ChronoBank Platform. * The official ChronoBank assets platform powering TIME and LHT tokens, and possibly other unknown tokens needed later. Platform uses MultiEventsHistory contract to keep events, so that in case it needs to be redeployed at some point, all the events keep appearing at the same place. * Every asset is meant to be used through a proxy contract. Only one proxy contract have access rights for a particular asset. * Features: transfers, allowances, supply adjustments, lost wallet access recovery. * Note: all the non constant functions return false instead of throwing in case if state change didn't happen yet.
